Aturan & Tata Tertib Ujian

1

Keamanan Browser Terdeteksi: Siswa dilarang keras membuka tab baru atau berpindah aplikasi. Aktivitas mencurigakan di luar jendela tes akan memicu sistem anti-cheat secara otomatis.

2

Batas Peringatan Pelanggaran: Apabila siswa terdeteksi meninggalkan layar ujian sebanyak 3 kali, sistem akan langsung mengirim seluruh jawaban terkunci dan mengakhiri sesi pengerjaan.

3

Penghitung Waktu Berjalan: Durasi pengerjaan akan segera dihitung ketika tombol "Mulai Ujian" ditekan. Silakan periksa sisa waktu pada bilah atas lembar tes.

Sinkronisasi Database Google Spreadsheet

Gunakan Google Spreadsheet sebagai basis data terdistribusi secara cuma-cuma.

Pastikan deploy sebagai Web App dengan izin akses "Anyone".

Langkah Penyediaan (Deployment):

1. Buat Google Spreadsheet Baru
Buat Spreadsheet kosong di Google Drive Anda.

2. Pasang Google Apps Script
Buka menu Extensions > Apps Script, kemudian salin kode di bawah ini lalu tempelkan di sana:

Klik pada box kode di atas untuk menyalin semua baris otomatis.

3. Deploy Web App
Klik Deploy > New deployment. Pilih jenis Web App. Set Execute as: Me dan Who has access: Anyone. Salin URL Web App yang dihasilkan lalu masukkan pada kolom input di atas.
